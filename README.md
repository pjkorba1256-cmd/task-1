# 🚀 Web Development Project

## 📌 Overview

This project is a modern web application built using **React + Vite + Tailwind CSS**. It demonstrates core frontend development concepts such as component-based architecture, responsive design, and interactive UI.

This project was created as part of a development task to showcase skills in:

* Frontend development
* UI/UX design
* Code structuring
* Deployment practices

---

## 🛠️ Tech Stack

* **React.js** – For building user interfaces
* **Vite** – Fast build tool and development server
* **Tailwind CSS** – Utility-first CSS framework for styling
* **JavaScript (ES6+)** – Core programming language
* **HTML5 & CSS3** – Structure and styling

---

## 📂 Project Structure

```
task-1/
│
├── public/            # Static assets
├── src/               # Main source code
│   ├── assets/        # Images and backgrounds
│   ├── components/    # Reusable UI components
│   ├── pages/         # Application pages
│   ├── App.jsx        # Main App component
│   ├── main.jsx       # Entry point
│
├── package.json       # Project dependencies
├── vite.config.js     # Vite configuration
└── README.md          # Project documentation
```

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/pjkorba1256-cmd/task-1.git

# Navigate into the project
cd task-1

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will run on:
👉 [http://localhost:5173/](http://localhost:5173/)
